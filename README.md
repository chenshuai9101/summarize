# Summarize 📄

> 企业级文本摘要平台 — AI Agent 技能

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-green)](https://github.com/chenshuai9101/summarize)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## 概述

支持多种内容源的智能摘要工具，包括URL、YouTube视频、播客、文章、PDF和本地文件。

## 核心功能

- **🔗 URL摘要** — 网页内容自动提取和摘要
- **🎬 视频转录** — YouTube/视频内容转文字并摘要
- **🎙️ 播客摘要** — 音频内容转录和要点提取
- **📑 PDF处理** — 本地PDF文件内容摘要
- **📝 长文精炼** — 长文本结构化摘要

## 使用方式

```bash
# 在 OpenClaw 中
/skill summarize
```
